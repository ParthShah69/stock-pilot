# Stock Portfolio Management System

A full-stack web application for managing stock portfolios with KYC verification, admin panel, and ML-powered stock predictions.

## Quick Setup

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
python app.py
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

## Features
- User authentication and KYC verification
- Portfolio management with buy/sell transactions
- Admin panel for KYC approval and query management
- ML-powered stock predictions using Prophet
- Real-time stock data integration
- Responsive React frontend
